CALENDAR STUFF AND COUNT THE DAYS
================

Calendar Stuff: This program can compare two dates to each other and figure out characteristics of a particular date.<br />
~ checks if a year is a leap <br />
~ checks how many days are in a given month <br />
~ checks if two dates are the same <br />
~ compares two dates <br />
~ checks if a date is valid <br />
~ gives the month given the integer <br />
~ gives the day given the integer <br />
~ checks the number of days between two dates <br />

Count the Days: uses the last method in Calendar Stuff to check the date between two dates (better UI)
